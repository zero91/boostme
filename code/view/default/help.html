如何使用Boostme

我们提醒您注意，您需要注册并登陆，才能享受我们的完整服务进行各项操作，否则您只有部分浏览的权限。

咨询区

致各位考研学子：

1、 寻找师兄师姐咨询：关于各位对于复试相关事宜（譬如复试时笔试内容、面试时常见的导师提问），如有任何疑虑，都可以在筛选框进行选择，譬如：您可以选择“北京”→“清华大学”→“化学系”→“化学”，然后便会出现清华大学化学系相关师兄师姐，点击其中一个进行付款，付款后您便可以获取师兄师姐的联系方式，这样您就可以尽情地咨询啦！

2、 评分机制：亲，咨询服务后，请您不要忘了对师兄师姐提供的咨询服务进行打分哦！这样不仅能保证您的权益，同时还能让我们为大家提供越来越优质的咨询服务！

致研究生师兄师姐们：

1、 提供咨询服务：各位研究生师兄师姐们是否想在闲暇时分赚一些外快呢？那就快来Boostme填写您所能提供到的帮助吧！您只要接一个电话就能帮助到苦恼中的广大考研学子！

2、 邀请机制（这是一个对师兄师姐来说不容错过的赚钱方式）：只要您的“提供咨询”申请通过了我们的审核，便可获得3个邀请码，您可以将这三个邀请码发给您的同学，只要他们提供了相应的咨询服务，您便可获得额外的收益哦！（受邀者前三次提供的服务您都可以分红哦！）

关于研究生师兄师姐如何套现？

首先，进入自己的主页，进入“余额套现”区，点击“认证”认证你的支付宝账号吧，只需提交1分钱的认证费，即可马上对您的支付宝进行认证(您需要再次登录支付宝确认收货哦！)。您可以通过认证后的账号对您所赚的资金进行套现，我们会在每周六晚对您的套现申请进行集体办理！

邀请用户

{$setting['site_name']}鼓励大家邀请研究生大牛们来注册,邀请别人来提供咨询是有现金奖励的哦！

什么是邀请码？

邀请码是系统赠送给提供咨询的用户的一个唯一的编号，用户使用此编号可以邀请身边的同学来注册。注册的同学成功使用该邀请码，则视邀请码拥有者成功邀请一人

什么是现金反馈？

用户使用Boostme邀请码成功邀请一个用户，如果被邀请的用户在Boostme上提供咨询，则每成功服务一考研同学，邀请者将获得现金奖励。

用户每成功邀请一个用户，且该用户提供了咨询服务，则邀请者将获得3次收入现金奖励的机会

资料区

1、下载资料：

按照您心仪院校的地区、名称、学院、专业进行搜索，这样就可以找到您心仪的资料了。如果还没有师兄师姐上传相应的资料，您可以去交流区发帖请求师兄师姐上传心仪的资料

2、上传资料：

如果您拥有珍贵的考研资料，您可以上传到资料区，并对该资料进行定价。好资料，大家齐分享！

3、收藏资料：

特别心仪清华大学建筑学专业？快快收藏吧！点击添加按钮将它添加到“快捷入口”就可以啦！

4、注意事项：

考研资料非常珍贵，师兄师姐也是自己花钱购买的，所以在分享资料时我们给予了他们充分的自主定价权，希望大家理解并支持。当然，我们保证定价处于合理区间，Boostme会严格审查每一份上传资料，请大家放心！
